# 📋 Call Tracker — Claude Code Skill

> Операционный ассистент руководителя: превращает транскрипцию встречи в задачи, сохраняет в Notion, рассылает команде в Telegram, отслеживает выполнение и делает документы по итогам встреч.

## Что делает

Один скилл с 7 режимами:

| Режим | Что делает |
|-------|------------|
| **ANALYZE** | Разбирает транскрипцию → вытаскивает все обязательства → создаёт карточки в Notion → рассылает в Telegram с кнопками |
| **STATUS** | Показывает срез: просроченные, ближайшие, в работе, выполненные |
| **REMIND** | Персональные напоминания в Telegram просроченным и по дедлайнам |
| **RESPONSES** | Обрабатывает ответы команды — кнопки и свободный текст, обновляет Notion |
| **REPORT** | Отчёт "что обещали — что сделали" с % исполнительности |
| **DOCUMENT** | Делает документ-артефакт: методичку, инструкцию, шаблон КП, конспект |
| **SETUP** | Первичная настройка Notion, Telegram-бота, Fireflies |

## Установка

### 1. Клонируй репозиторий

```bash
cd ~/.claude/skills/
git clone https://github.com/YOUR_USERNAME/call-tracker.git
```

Или скачай `SKILL.md` вручную и положи в `~/.claude/skills/call-tracker/SKILL.md`.

### 2. Подключи интеграции (см. полную инструкцию)

- **Notion MCP** — для хранения задач
- **Telegram MCP** — для рассылки команде
- **Fireflies MCP** _(опционально)_ — для автоматического получения транскриптов

### 3. Первый запуск

Открой Claude Code, скажи:

```
Разбери встречу. Вот транскрипция: [вставь текст]
```

Скилл автоматически запустится, спросит только критичное и сам создаст задачи.
